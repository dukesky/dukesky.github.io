Spoken Language Detection

This project aimed to detect language spoken in the audio record.

In this priject, I had 60 audio records (10 min for each) which contains reading in English, Mandarin and Hindi. My job is training a deep learning model to recognize spoken language based on this 60 audio data. I implemented my model by the following step:

%nbsp; %nbsp; 1. I segmentate each audio record into small blocks which contain 10 seconds of language spoken audio. This expand my dataset into 3,000 audio data

%nbsp; %nbsp; 2. I used Python pakage LibROSA to extract fetures from audio. This pakage help me extract 64 features for each second using Mel-frequency cepstral coefficients (MFCCs)

%nbsp; %nbsp; 3. Build Recurrent Nerual Network(RNN) model with 2 LSTM layers. There model obtained an accuracy of 73.5%
